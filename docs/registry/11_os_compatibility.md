# 11_os_compatibility

- Backend runtime: Python 3.12.
- Frontend build runtime: Node.js 20+, npm.
- Режимы:
  - Yandex Cloud serverless (основной);
  - local/containerized verification через CLI и smoke scripts.

План тестирования совместимости:
- Ubuntu LTS (доверенный Linux дистрибутив);
- Astra Linux/ALT Linux (плановая проверка в российском контуре).

Сертифицированная совместимость не заявляется без фактических протоколов тестирования.
