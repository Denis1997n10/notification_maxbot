# 01_product_description

Этот документ описывает соответствующий аспект подготовки продукта к реестровой подаче.

- Продукт: платформа уведомлений жителей (ПО, не ПАК).
- Развертывание: Yandex Cloud ru-central1, Cloud Functions/API Gateway/YDB/Lockbox/Object Storage.
- Сборка воспроизводима через локальные скрипты и CI, включая self-hosted runner.
- Для production допустим контур без GitHub-hosted runner.

## Практические положения
- Исходный код хранится в git-репозитории и может зеркалироваться в российской инфраструктуре.
- Объектный код и релизные артефакты размещаются в Object Storage.
- Секреты хранятся в Lockbox; в релизы и tfvars не включаются.
- Эксплуатационные проверки: health/version API, smoke-тесты, CLI проверок.
